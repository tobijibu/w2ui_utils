w2utils.unlock

unlock(box)

HTML要素のロックを解除します。 戻り値はundefinedです。

box : HTML要素, jQueryオブジェクト
ロックを解除する要素を指定します。

説明

この関数は.lock()でロックされたboxのロック状態を解除します。

このメソッドはGrid、Sidebar、Layout、Popupで呼び出され、コンテンツをアンロックします。

デモ

"ロック解除"ボタンをクリックすると、ロック中の要素が解除されます。

ロック中

JavaScript

w2utils.lock($('#demo1'));
$('#lock').on('click', function(){
  $('#demo1').text('ロック中');
  w2utils.lock($('#demo1'));
});
$('#unlock').on('click', function(){
  $('#demo1').text('ロック解除');
  w2utils.unlock($('#demo1'));
});

CSS

#demo1 {
  position: relative;
  height: 160px;
  margin-bottom: 8px;
}

HTML

<div id="demo1">ロック中</div>
<button id="lock">ロック</button>
<button id="unlock">ロック解除</button>
