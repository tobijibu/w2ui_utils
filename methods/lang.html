w2utils.lang

lang(phrase)

phraseを翻訳します。 戻り値はstringです。

phrase : string
翻訳するフレーズ。

説明

翻訳が用意されているなら、この関数でフレーズを翻訳することができます。 全ての翻訳は.settings.phrasesに格納されており、.locale()メソッドでロードされます。

以下のように指定することができます。

var a = w2utils.lang('Today');
console.log(a);

デモ

'Today'を翻訳して、結果をconsole.logに出力します。
本来、このデモではja-jp.jsonは必要ありません。 既存のフレーズを変更する場合はja-jp.jsonをダウンロードする必要があります。
https://github.com/vitmalina/w2ui/tree/master/src/locale

JavaScript

var a = w2utils.lang('Today');
console.log(a);   // "Today"
 
w2utils.locale('../json/ja-jp.json');
// フレーズを追加
var phrases = {
  "Today": "今日"
};
$.extend(w2utils.settings.phrases, phrases);
a = w2utils.lang('Today');
console.log(a);   // "今日"
