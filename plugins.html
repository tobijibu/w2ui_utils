Plugins

w2uiにはいくつかの共通機能を持ったjQueryのプラグインがあります。 プラグインにはウィジェットのレンダリング、破棄、 テキストハイライトコンテキストメニュー生成等の機能があります。

$().w2render(name)

前に生成された要素をレンダリングするプラグインです。 例えば、表示中のグリッドを別のコンテナにレンダリングしたい場合、 以下のように指定します。 

w2ui['myGrid'].render($('#id')[0]); // "#id"が無ければエラーになります。
OR 
$('#id').w2render('myGrid'); // "#id"が無くてもエラーになりません。

どちらも同じ動作ですが、2つ目の方が分かりやすいでしょう。


$().w2destroy(name)

前に生成された要素を破棄するプラグインです。 w2uiコントロールを破棄し、w2uiオブジェクトから削除します。 w2uiコントロールが存在しない場合は何もしません。

$().w2marker(str)

テキストをハイライトするプラグインです。Grid内で内部的に使用されます。

$('#id').w2marker('text');  // 要素内のテキストを全てハイライトします。
