Events

w2uiは全て同じイベントフローを持っています。 イベントを追加/削除するには以下の2つの方法があります。

  • Eventプロパティで使う。
  • .on()、.off()メソッドを使う。

1つ目の方法はシンプルで、w2uiオブジェクト生成時に指定できますが、 1つのイベントに対して指定可能なイベントリスナーは1つだけです。 2つ目の方法では、複数のイベントリスナーを追加可能ですが、ランタイム時にだけ利用可能です。 これにより、イベントフローを細かく指定することができます。 "on"ではじまるプロパティは全てイベントです。

プロパティ、メソッド

handlers

イベントハンドラの配列

on

w2uiオブジェクトにイベントハンドラを追加する。

off

w2uiオブジェクトからイベントハンドラを削除する。

trigger

.on()で追加した全てのイベントを実行する。

イベントハンドラ

イベントが動きだすと、ハンドラはfunction (event)のように1つのパラメータを受け取ります。 このeventパラメータはイベント情報が含まれたオブジェクトです。

イベントハンドラは以下のように指定します。

function (event) {
    // event actions
}

全てのイベントハンドラは、デフォルトの動作前に発生します。 デフォルトの動作をキャンセルしたい場合は以下のように指定します。 

function (event) {
    event.preventDefault(); // cancel event
}

デフォルトの動作の後に実行したい場合は以下のように指定します。

function (event) {
    event.done(function () {
        // event actions
    });
}
