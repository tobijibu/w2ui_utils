common.handlers

更新日: 2018.09.07

イベントハンドラの配列です。

Array, default = []

イベントハンドラを指定する場合に、この配列を直接変更する必要はありません。 イベントを追加する場合は.onメソッド、イベントを削除する場合は.offメソッドを使います。 追加したイベントハンドラを確認する為に、この配列をループ処理すると便利です。 

$('#layout').w2layout({
    name    : 'layout',
    panels  : [
        { type: 'top', size: 40 },
        { type: 'preview', size: 200 }
    ]
});

// add event listener for all events
w2ui['layout'].on('*', function (target, eventData) {
    console.log(target);
    console.log(eventData);
});

// iterating through event handlers
for (var i in w2ui['layout'].handlers) {
    console.log(w2ui['layout'].handlers[i]);
}
* w2ui['layout'].on('*', function(){});で全てのトリガーに対してイベントリスナーを設定できるようなのですが、 resizeだけが登録され、他のイベントは登録されないようです。(要確認)
